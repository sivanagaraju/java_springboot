# 02 — Gradle Advanced

## Overview

This sub-module covers advanced Gradle concepts needed for production-grade Spring Boot applications. You will learn how to write custom tasks, configure build profiles for different environments, and understand the internals of the Spring Boot Gradle Plugin.

## Core Concepts Covered

1. **Custom Tasks** — Writing Groovy-based tasks for code generation, environment checks, and build automation
2. **Build Profiles** — Configuring dev/test/prod environments via Gradle properties and Spring profiles
3. **Spring Boot Gradle Plugin** — Understanding `bootJar` vs `jar`, `bootRun` configuration, and `bootBuildImage`

## Structure

- `explanation/` — Deep-dive markdown files for each advanced concept
- `exercises/` — Hands-on tasks to build custom Gradle automation
