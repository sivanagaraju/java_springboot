# Spring Boot Architecture Top Resource Guide

This file is for outside resources only: books, official docs, blogs, and videos for Spring container and Boot architecture learning.

## Books

| Resource | Why it helps |
|---|---|
| Spring Start Here by Laurentiu Spilca | very good for IoC and container mental models |
| Spring in Action | broad Spring coverage with practical architectural examples |
| Pro Spring Boot | helpful when you want deeper Boot-specific startup and configuration detail |

## Official Docs

| Resource | Why it helps |
|---|---|
| Spring Framework Core Container docs | primary source for beans, scopes, lifecycle, and DI |
| Spring Boot Reference Guide | primary source for auto-configuration, profiles, and externalized config |
| Spring Boot Actuator docs | shows how to inspect what Boot wired at runtime |

## Blogs and Articles

| Resource | Why it helps |
|---|---|
| Baeldung Spring articles | useful for targeted explanations of IoC, profiles, and conditions |
| Spring blog | good for official architectural guidance and release-driven context |
| reflectoring.io Spring articles | practical explanations of Boot mechanics and clean layering |

## Videos

| Resource | Why it helps |
|---|---|
| Java Brains Spring playlists | good for early-stage Spring architecture understanding |
| official Spring I/O talks | strong for container internals and Boot design direction |
| Dan Vega Spring Boot videos | approachable Boot-focused walkthroughs |

## Suggested Order

1. Learn IoC and bean lifecycle first.
2. Then study Boot startup and auto-configuration.
3. Use blog posts for specific architecture questions after the fundamentals click.

## Interview Questions

1. Which external resource would you use first to understand Boot auto-configuration deeply, and why?
2. Why is the official Boot reference more important than random snippets when learning architectural behavior?
