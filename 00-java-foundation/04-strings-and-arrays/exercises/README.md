# Strings & Arrays — Exercises

## Prerequisites
Read all explanation files in `../explanation/` before attempting.

## Exercises

### Ex01 — String Manipulation (`Ex01_StringManipulation.java`)
Implement: `reverseString`, `countVowels`, `isPalindrome`, `compressString`, `areAnagrams`. No library shortcuts.

**Skills:** `StringBuilder`, `charAt()`, `Arrays.sort()`, string algorithms
**Time estimate:** 30 minutes

### Ex02 — Matrix Operations (`Ex02_MatrixOperations.java`)
Implement: `transposeMatrix`, `rotateMatrix90`, `spiralOrder`, `findSaddlePoint` on 2D int arrays.

**Skills:** 2D arrays, index math, in-place algorithms
**Time estimate:** 40 minutes
