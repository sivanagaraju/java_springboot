# 01 - Java Basics Exercises

These exercises are designed to test your understanding of primitive types, variables, operators, and control flow in Java.

## Structure

- **Ex01_TypeConversion.java**: Tests your knowledge of when Java allows implicit conversion (widening) and when it requires explicit casting (narrowing), along with potential data loss traps.
- **Ex02_Calculator.java**: Builds a simple console-based calculator to practice `Scanner` for I/O, `switch` expressions for routing math operations, and `while` loops for continuous execution.

## How to Run

Navigate to the project root and use the Gradle wrap to execute a specific class:

```bash
./gradlew :00-java-foundation:run --args="Ex01_TypeConversion"
./gradlew :00-java-foundation:run --args="Ex02_Calculator"
```
