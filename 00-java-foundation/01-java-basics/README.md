# 01 - Java Basics

Welcome to the foundation of Java. If you are coming from Python, this module represents the biggest paradigm shift: moving from a dynamic, interpreted execution model to a strict, statically-typed, and compiled execution model.

## Learning Objectives
1. Understand the Java execution pipeline (Source -> Bytecode -> JVM -> Native).
2. Differentiate between primitive memory structures and object memory structures.
3. Master Java's logical and relational operators, especially avoiding the `==` vs `.equals()` trap for objects.
4. Write structured loops and conditions using Java 14+ idioms.

## Directory Structure
- `explanation/`: Theory-focused Markdown guides and corresponding `.java` execution demos.
- `exercises/`: Practical challenge files to test your compilation and syntax skills.

## Study Path
Read the markdown files and run the corresponding Java demos in this order:
1. `01-how-java-works.md` -> `HowJavaWorks.java`
2. `02-variables-datatypes.md` -> `VariablesDemo.java`
3. `03-operators.md` -> `OperatorsDemo.java`
4. `04-control-flow.md` -> `ControlFlowDemo.java`
5. Wrap up by completing `Ex01` and `Ex02` in the `exercises` directory.

Review the `MINDMAP.md` to visualize how everything connects.
