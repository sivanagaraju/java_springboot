# Module 5: Collections Framework Exercises

This directory contains exercises designed to test your understanding of Java Collections, explicitly forcing you to choose the correct Data Structure based on Big O performance and logical constraints.

## Exercise 1: Inventory Management (`Ex01_InventoryManagement.java`)
**Goal:** Practice `List` and `Set` mechanics.
**Tasks:**
1. Create an `ArrayList` to store an inventory log (where duplicates are allowed because items can be scanned multiple times).
2. Create a `HashSet` to extract only the *unique* item names from the inventory log.
3. Write a method to check if a specific item exists in the unique set.

## Exercise 2: Student Grades (`Ex02_StudentGrades.java`)
**Goal:** Practice `Map` and `PriorityQueue` mechanics.
**Tasks:**
1. Build a `HashMap` mapping a Student's Name (String) to their Grade (Integer).
2. Add several students and their grades.
3. Write logic to look up a student's grade efficiently.
4. (Bonus): Use a `PriorityQueue` to store the grades and easily poll the highest (or lowest) grade in the class.
