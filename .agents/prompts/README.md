# Project-level Prompts

This folder is reserved for reusable prompt templates for specialized tasks in this repository.

## 🗂 Planned Templates
Templates will be added here as the project matures. Examples of future templates:
- **Content creation:** Standardized prompts for writing explanation `.md` files with all 3 content standards.
- **Code review:** Prompts for auditing Java code against the 4-layer commenting standard.
- **Python-to-Java:** Prompts for comparing Python/FastAPI patterns to Spring Boot equivalents.
